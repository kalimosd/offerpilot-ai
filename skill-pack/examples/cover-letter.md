# Cover Letter Example

## Task

Generate a cover letter from a resume and a job description.

## Inputs

- original resume
- job description
- target language

## Workflow

1. identify the most relevant experiences in the resume
2. map them to the job description
3. write a concise letter with specific evidence
4. remove filler and generic claims

## Expected Output

- concise and credible tone
- direct relevance to the role
- no unsupported claims
