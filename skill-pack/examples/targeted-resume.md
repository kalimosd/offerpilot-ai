# Targeted Resume Example

## Task

Rewrite a resume so it better matches a specific job description.

## Inputs

- original resume
- target job description
- target language

## Workflow

1. extract the key role requirements from the job description
2. identify the most relevant experiences in the resume
3. reorder or rephrase emphasis without changing the facts
4. check whether role titles, technologies, and business impact still match the source

## Expected Output

- stronger relevance to the target role
- natural use of keywords from the job description
- no fabricated alignment
